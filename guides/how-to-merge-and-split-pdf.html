How to Merge & Split PDF Free Online (Nothing Uploaded)

You need to combine a few PDFs into one file, or pull a handful of pages out of a big document. Most "free" PDF sites make you upload the file to their servers first. Here's how to do both — merge and split — entirely in your browser, so the document never leaves your device.

Why "nothing uploaded" actually matters

PDFs are where the sensitive stuff lives: signed contracts, bank statements, medical forms, tax returns, ID scans. The typical online PDF tool asks you to upload that file to a remote server, where it's processed and — you hope — deleted afterward. You have no way to verify what happens to it, and for anything confidential that's a real risk.

The tools on this site work differently. They use pdf-lib, an open-source JavaScript library that reads and rebuilds PDFs right inside the browser tab. When you pick a file, it's loaded into your browser's memory with the standard FileReader API and processed locally. There's no upload step, no server, and no account. You can even turn off your internet connection after the page loads and every function still works.

How to merge PDF files into one

Merging is the answer when you have several separate PDFs — say a cover letter, a résumé, and a portfolio — that should arrive as a single document.

  1. Open the Merge PDF tool.
  2. Drop all the PDFs onto the box, or click to browse and select several at once. Each file appears in a list with its page count.
  3. Put them in the right order using the ▲▼ buttons. The order in the list is the order the pages will appear in the final document.
  4. Click Merge PDF. The combined file downloads instantly as merged.pdf.

Because the pages are copied rather than re-rendered, nothing about their appearance changes — text stays selectable, images stay sharp, and vector graphics stay crisp. Merging simply stitches the pages together into one file.

How to split a PDF or extract pages

Splitting is the reverse: you have one big PDF and only need part of it. The Split PDF tool gives you two ways to do that.

Extract a page range

This pulls the pages you name into a brand-new PDF. After you drop your file in, choose Extract a page range and type the pages you want. The syntax is simple: use commas for individual pages and dashes for ranges. For example, 1-3, 5, 8-10 produces a PDF containing pages 1, 2, 3, 5, 8, 9, and 10, in that order. Leave the box blank to keep every page. Click Split PDF and the new file downloads as yourfile_extract.pdf.

Split every page into its own file

When you need each page as a standalone document — for filing, or to send pages to different people — choose Split every page. The tool creates one single-page PDF per page and bundles them into a ZIP, named like yourfile_page_01.pdf, yourfile_page_02.pdf, and so on. It downloads as a single ZIP so you don't get flooded with individual download prompts.

A note on password-protected PDFs

If a PDF is encrypted with a password, it has to be unlocked before it can be merged or split — that's true of any tool, because the file's contents are scrambled until the password is supplied. Open the file in your usual PDF viewer, remove the protection (often via Print → Save as PDF or a "remove password" option), then run the unlocked copy through the tool here.

Frequently asked questions

Is it really free, with no upload?

Yes. There's no signup, no watermark, and no upload. Both the merge and split tools run entirely in your browser, so your PDFs never touch a server.

Is there a file size or page limit?

There's no hard limit. Since everything runs on your device, very large PDFs simply use more memory and take a moment longer to process.

Will merging or splitting change the quality?

No. Pages are copied intact, so text, images, and vector content are preserved exactly. The tools rearrange and re-package pages; they don't re-render or recompress them.
