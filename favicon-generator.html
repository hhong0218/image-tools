What is a favicon — and why do I need all these sizes?

      
A favicon is the small icon shown in a browser tab, bookmarks, and history. Modern sites also need larger versions for the Apple touch icon (when someone adds your site to an iPhone home screen) and Android/PWA icons (192×192 and 512×512, used for home-screen shortcuts and installable web apps). This tool generates every common size from a single image so you don't have to resize each one by hand.

Frequently asked questions

      
Is my image uploaded anywhere?

      
No. Every size is generated locally with an HTML <canvas>, and packaged into a ZIP with the open-source JSZip library — entirely in your browser. Nothing is uploaded, there's no server and no account.

      
Do I get a real .ico file?

      
This tool generates PNG files at every standard size (which every modern browser accepts directly via <link rel="icon">) rather than a legacy multi-resolution .ico container. Using PNGs with the provided HTML snippet is the modern, widely-supported approach and avoids needing an ICO-specific encoder.

      
What image should I start with?

      
A square image with simple shapes and enough contrast to still read clearly at 16×16px. Logos, initials, or a simple icon work much better than a busy photo, which turns into a blur at favicon size.
